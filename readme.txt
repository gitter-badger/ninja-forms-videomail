== Description ==

A Videomail integration for Ninja Forms.

== Installation ==

This section describes how to install the plugin.

1. Upload the `ninja-forms-videomail` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Requires the Ninja Forms plugin.

== Use ==

This plugin adds a new field type, videomail, to the form builder.

The videomail field type adds a videomail video capture to the form display.
